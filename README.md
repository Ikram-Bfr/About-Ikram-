I am a PhD researcher in Artificial Intelligence & Engineering Faculty at Multimedia University, working on AI-driven optimization for next-generation edge networks. My research focuses on intelligent task offloading, dynamic resource allocation, reinforcement learning, digital twin integration, and cybersecurity for Internet of Vehicles (IoV), Mobile Edge Computing (MEC), electric vehicle charging systems, and connected edge environments.

I enjoy building scalable AI frameworks that connect theory, simulation, and deployable intelligent systems.


## Research Focus

- AI-driven task offloading and resource allocation in MEC and IoV systems
- Deep reinforcement learning for decentralized decision-making
- Digital twin-enabled optimization for edge and vehicular networks
- Temporal Convolutional Networks for cyberattack detection
- Secure Open Charge Point Protocol (OCPP) for EV charging systems
- STAR-RIS-assisted UAV systems for integrated sensing, communication, and computation
- Sensing and perception for IoV, V2X, and V2I systems.

## Featured Work

### Advanced TCN Framework for EV Charging Cybersecurity

Designed an adaptive Temporal Convolutional Network framework for scalable cyberattack detection in electric vehicle charging stations.

**Keywords:** TCN, EV charging, cybersecurity, adaptive dilation, anomaly detection

### Secure OCPP With Temporal Convolutional Networks

Developed a deep learning approach for securing Open Charge Point Protocol communication in electric vehicle charging environments.

**Keywords:** OCPP, EV infrastructure, TCN, cyberattack detection, edge security

### Reinforcement Learning-Driven STAR-RIS UAV Network

Built a reinforcement learning-driven framework for STAR-RIS-assisted UAV systems supporting integrated sensing, communication, and computation.

**Keywords:** reinforcement learning, STAR-RIS, UAV, ISCC, resource allocation
